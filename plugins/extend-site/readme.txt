== Description ==
Extend Site is an essential toolkit for WordPress.
It provides useful features such as custom post types, widgets, and site extensions to enhance your WordPress experience.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/extend-site` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.

== Frequently Asked Questions ==

= Can I use this plugin with any theme? =
Yes. Extend Site is designed to work with any well-coded WordPress theme.
